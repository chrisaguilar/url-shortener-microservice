Free Code Camp Back End Project #3 - URL Shortener Microservice

Usage
curl -X POST -H "Content-Type: application/json" -d '{"url": "https://google.com"}' https://chrismaguilar.com/url-shortener/api


Output
{
    "$id": "rkmiLDldf",
    "$url": "https://google.com"
}



Visiting https://chrismaguilar.com/url-shortener/api/rkmiLDldf will redirect you to https://google.com
